# Lista de la compra

App web muy simple para gestionar la compra de la semana que viene.

## Qué hace

- Al abrir la app puedes escribir y añadir productos directamente.
- Cada producto se puede eliminar de forma individual.
- El botón `Compra hecha` vacía toda la lista de la semana siguiente.
- Los datos se guardan en Supabase.

## Configuración en Supabase

1. Abre el SQL Editor de tu proyecto `gqhprdaattnetouyqpbm`.
2. Ejecuta el contenido de [`supabase-setup.sql`](./supabase-setup.sql).

## Cómo abrirla

Puedes abrir [`index.html`](./index.html) directamente en el navegador o servir la carpeta con un servidor estático.
